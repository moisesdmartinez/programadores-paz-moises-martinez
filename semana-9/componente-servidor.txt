Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del sistema y permite ejecutar aplicaciones.

Node.js:
Es un entorno que permite ejecutar JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de actividad y eventos del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
